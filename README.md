This project presents an interactive Bank Loan Report Dashboard designed to provide comprehensive insights into loan application and performance data. The dashboard facilitates real-time monitoring of key financial metrics, loan statuses, and applicant demographics, enabling data-driven decision-making for risk assessment and portfolio management.

‚ú® Key Features
Summary View: Quick overview of total loan applications, funded amounts, amounts received, average interest rate, and Debt-to-Income (DTI) ratio.

Performance Metrics: Detailed breakdown of Good Loans vs. Bad Loans with associated amounts and percentages.

Loan Status Analysis: A table showing performance based on Current, Charged Off, and Fully Paid statuses.

Historical Trend: Visualization of Total Loan Applications by Month to track volume trends. * Geographical Analysis: Distribution of Total Loan Applications by State (US map view). * Demographic Insights: Analysis of applications by Loan Purpose, Home Ownership, Employee Length, and Loan Term.

Detailed Data: A sortable and filterable Details view of individual loan records, including Issue Date, Funded Amount, Interest Rate, and Installment.

üìä Metrics Tracked
The dashboard focuses on the following primary metrics:

Total Loan Applications: Overall volume of applications (e.g., 38.6K MTD).

Total Funded Amount: The total capital disbursed to applicants (e.g., $435.8M MTD).

Total Amount Received: The total payments collected from borrowers (e.g., $473.1M MTD).

Avg Interest Rate: The mean interest rate across all active loans (e.g., 12.0%).

Avg DTI: The average Debt-to-Income ratio of applicants (e.g., 13.3%).

Good Loan Issued: Percentage of loans considered "Good" (e.g., 86.2%).

Bad Loan Issued: Percentage of loans considered "Bad" (e.g., 13.8%).
